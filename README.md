# Rolo-Pong
## Alr, at the moment im writing this readme, my project is made in python, and i used Tkinter, random, turtle and time to make the main functions.
## The game includes a score board that saves the current score of the match, which ends when one player reaches 10 points, an infinite 1v1 match (can exit by pausing and clicking on the quit button) and a player VS CPU mode, that features matches against an easy, medium and hard bots!!
## I suggest to play the game in full screen for a better experience.
# CONTROLS
# In main menu
## Click on any of the modes to chose it, but the match starts almost instantly so be careful.
# While a match
## Press down the space bar to pause the game in any of the modes and exit to the main menu.
## For the red player: W to move upwards, S to move downwards
## For the blue player: Up arrow to move upwards, down arrow to move downwards
# How i used the libraries
## I used tkinter just to print the message at the end, and for the easter egg that appears when the game score is 6-7 and 7-6
## I used turtle to create the movement of both paddles and the ball.
## I used time to make a better playing experience by making the overall game more fluid
## I used random to randomize the starting movements of the ball
